Este proyecto es una réplica de la interfaz de usuario de Spotify, creado con el objetivo de practicar y mejorar habilidades en:

- Desarrollo de aplicaciones con Astro
- Integración de React con Astro
- Uso de TypeScript para tipado estático
- Optimización de rendimiento y SEO
- Diseño responsive y UI/UX
- Reproducción de audio web

Tecnologias usadas: 
    - **Astro** - Framework web moderno para sitios rápidos
    - **React** - Para componentes interactivos
    - **TypeScript** - Superset de JavaScript con tipado estático
    - **Node.JS** - Utilizado para realizar las conexiones API
    - **Tailwind CSS** - Framework de CSS utility-first
    - **Spotify Web API** - Integración con servicios de Spotify

Este proyecto fue desarrollado como ejercicio de práctica para:

- Reforzar conocimientos en React, TypeScript y Node.js
- Implementar patrones de diseño modernos
- Mejorar habilidades de integración con APIs
- Practicar desarrollo de interfaces complejas
- Fortalecer el portfolio de desarrollo frontend y backend
